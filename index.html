Windows Subsystem for Linux 2 (WSL2)

The WSL2 requires Windows 10 1903+ with Build 18362.1049+ or 18363.1049+ in 64-bit version (WSL Kernel Update can't be updated in 32-Bit version)

You can see actual version and build pressing Win + R, then write winver and press Enter.

  1. Activate Windows Subsystem for Linux. Run next command from PowerShell with administrative rights. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

  2. Activate the Virtual Machine Platform. May need to enable this CPU option in BIOS. Run next command from PowerShell with administrative rights.dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

  3. In windows 10 the "hypervisor launch type" is not configured to working WSL2, you can see the actual status with bcdedit (Run in terminal as administrator) if "hypervisorlaunchtype off" are displayed, then run: BCDEDIT /Set {current} hypervisorlaunchtype auto(Run in terminal as administrator)

  4. >>>> At this point a computer restart is required. <<<<

  5. Download from here the Windows Subsystem for Linux 2 kernel Update. Then install it.

  6. Run command from PowerShell with administrative rights.wsl --set-default-version 2

  7. Open the Microsoft Store and search your favorite Linux distribution or download the Ubuntu 20.04 LTS package. Then open downloaded file (Ubuntu_2004.2020.424.0_x64.appx) or run from PowerShell with administrative rights:

    Add-AppxPackage c:\path\to\Ubuntu_2004.2020.424.0_x64.appx

If you want, a PowerShell script can be used to make these changes and downloads. First of all run set-executionPolicy -ExecutionPolicy Unrestricted -force from elevated permissions Powershell window, then Just type in PowerShell & c:\path\to\update_WSL2.ps1 or right click on file and click on "run with PowerShell" (Administrative permissions ask when runnig the script)

    Linux commands

  • Open Linux terminal to set user name and password.

  • Change home (~) Directory:

    1. Run in terminal: sudo vim /etc/passwd
    2. Search for username:x:1000:1000:"",,,:/home/<username>:/bin/bash
    3. then change /home/<username> for /mnt/d or wanted folder
    4. Save file and exit from vim pressing Esc and writing :wq, press Enter.
    5. Exit from terminal with logout
    6. To verify Enter again to terminal and writing cd ~ and press Enter
    7. Then write pwdand press Enter
    8. Verify the folder showing are the selected in ths example: /mnt/d
  • Run sudo apt update to retrieve the list of latest updates
  • Run sudo apt upgrade to upgrade Linux to the latest version.
  • Run sudo apt install nodejs to install node.js
  • Run sudo apt install npm to install Node Package Manager
  • Run git --version to verify git installed version. Compare with the git download webage
  • Run sudo apt install ruby-full to install Ruby on rails
  • Run sudo gem install compass to install compass preprocessor

I forked his repositories to fix a number issues I add some features I had wanted. I also wanted to improve the efficiency of the matching.
